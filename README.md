## Introduction 📖
**CRUD User Management** provides basic user management features, serving as a solid starting point for learning or developing more complex PHP-based applications.

## Features ✨
- ➕ Add new users
- ✏️ Edit user details
- ❌ Delete users
- 📋 View a list of users
